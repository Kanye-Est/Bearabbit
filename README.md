# 🐰❤️🐻 Bearabbit Love Game

给女朋友的小惊喜游戏！

## 🎮 游戏说明

- 控制可爱的小兔子/小熊收集掉落的爱心和花朵
- 使用键盘左右箭头键、A/D 键，或者在手机上触摸拖动
- 收集 200 分就能通关，获得专属鼓励！

## 📱 让她扫码打开的方法

### 方法一：使用 Vercel（推荐，免费且简单）

1. 安装 Vercel CLI：
   ```bash
   npm install -g vercel
   ```

2. 在 bearabbit 文件夹中运行：
   ```bash
   vercel
   ```

3. 按提示操作，部署完成后会得到一个链接

4. 用任意二维码生成器（如 [qr-code-generator.com](https://www.qr-code-generator.com/)）把链接转成二维码

### 方法二：使用 GitHub Pages

1. 创建 GitHub 仓库，把 `index.html` 上传上去

2. 在仓库设置中开启 GitHub Pages

3. 获得链接后生成二维码

### 方法三：使用 Netlify Drop（最简单）

1. 访问 [app.netlify.com/drop](https://app.netlify.com/drop)

2. 直接把 `bearabbit` 文件夹拖进去

3. 几秒钟后就能得到链接，然后生成二维码
